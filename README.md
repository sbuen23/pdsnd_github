### Project Title
Explore US Bikeshare Data

### Description
This project analyzes bikeshare usage data from major US cities as part of Udacity's Git and Version Control course.  
The goal is to demonstrate the use of Git and GitHub through branching, committing, merging, and maintaining project documentation.

The repository contains:
- A Python script (`bikeshare.py`) used to explore bikeshare data
- A `.gitignore` file to exclude data files (CSV) from version control
- Separate branches dedicated to documentation and refactoring, created to simulate a professional Git workflow

### Files used
- `bikeshare.py`  
- `.gitignore`  
- `new_york_city.csv` (used locally only, not uploaded to GitHub)
