### Project Title
Explore US Bikeshare Data

### Description
This project analyzes bikeshare data from three major US cities using Python.  
The purpose is to practice Git and GitHub workflows, including branching, commits, merging, and documentation.  
The repository includes:
- A Python script (`bikeshare.py`)
- A `.gitignore` file to prevent uploading CSV data
- Separate branches for documentation and refactoring

### Files used
- `bikeshare.py`  
- `.gitignore`  
- `new_york_city.csv` (used locally only, not uploaded to GitHub
